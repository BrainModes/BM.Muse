Detailed Description

A IXNMuse associated with a IXNMuseFileReader

This interface may be used to play back packets from a .muse file as MuseDataPackets.

See also
IXNReaderMuseBuilder, IXNReaderMusePlaybackSettings

Method Documentation

- (nullable IXNMuse *) asMuse

Downcast to IXNMuse

Note that most of the API of the returned Muse is stubbed and will cause assert failures if called in debug mode. The methods that are implemented are:

  • isLowEnergy
  • register*
  • unregister*

Note that the only listener that will ever receive packets from a IXNReaderMuse is IXNMuseDataListener.

- (void) playback

Play back the IXNMuseFile and call each listener for each packet as appropriate.

The speed of the playback is controlled by the playback settings. Simulated playback settings require the IXNReaderMuse to be constructed with an IXNEventLoop to simulate the time between packets. Calling playback without an IXNEventLoop will not playback the file and will log a warning.

This only works once; the file is not rewound on subsequent calls. If you want to play a file again, create another IXNReaderMuse

See also
IXNReaderMusePlaybackSettings
- (void) run

Run through all packets in the file, calling each listener for each packet as appropriate.

This only works once; the file is not rewound on subsequent calls. If you want to play a file again, create another IXNReaderMuse

Deprecated:
Use playback: with the playback setting IXNReaderMusePlaybackSettingsAsFastAsPossibleWithSavedTimestamp to replicate this behaviour.
- (void) runInRealTimespan

Run through all packets in the file, calling each listener for each packet as appropriate in 1:1 timespan

This only works once; the file is not rewound on subsequent calls. If you want to play a file again, create another IXNReaderMuse

Deprecated:
Use playback: with the playback setting IXNReaderMusePlaybackSettingsSimulatedWithSavedTimestamp to replicate this behaviour.
