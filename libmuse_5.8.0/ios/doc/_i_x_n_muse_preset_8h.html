Detailed Description

Defines all possible Muse presets. See this link for details: https://sites.google.com/a/interaxon.ca/muse-developer-site/museio/presets

Presets are specific to the model of headband. For example, setting a preset that is intended for Muse 2014 ( MU_01 ) headband on a Muse 2016 ( MU_02 ) headband will not work. Attempting to set an incorrect preset on a headband will result in a warning in the log and the incorrect preset will be ignored. To avoid this, you should first connect to the headband without setting a preset and check the IXNMuseConfiguration to get the model of the headband and then set the appropriate preset. You can also obtain the current preset of the headband from the IXNMuseConfiguration

If you change the preset to while the headband is connected, the headband will disconnect. If the preset is valid for that headband model, then the headband will automatically reconnect. If the preset is invalid for that headband model, then the headband will remain disconnected.

Enumerator
IXNMusePresetPreset10 

4 CH, 10 bit @ 220 sps, compression ON, no accelerometer, no battery data, no error data, no DRL/REF data.

Availability: Muse 2014 ( MU_01 ) only

IXNMusePresetPreset12 

4 CH, 10 bit @ 220 sps, compression ON, 50 sps accelerometer data, 0.1 sps battery data, no error data, no DRL/REF data

Availability: Muse 2014 ( MU_01 ) only

IXNMusePresetPreset14 

4 CH, 10 bit @ 220 sps, compression ON, 50 sps accelerometer data, 0.1 sps battery data, real-time error data, 10 bit @ 10 sps DRL/REF data

This is the default for Muse 2014 ( MU_01 ).

Availability: Muse 2014 ( MU_01 ) only

IXNMusePresetPreset20 

5 CH, 12 bit @ 256 sps, 52 sps accelerometer/gyro, 0.1 sps battery, 32 sps DRL/REF.

Availability: Muse 2016 ( MU_02 ) only

IXNMusePresetPreset21 

4 CH, 12 bit @ 256 sps, 52 sps accelerometer/gyro, 0.1 sps battery, 32 sps DRL/REF.

This is the default for Muse 2016 ( MU_02 ) only

Availability: Muse 2016 ( MU_02 ) only

IXNMusePresetPreset22 

4 CH, 12 bit @ 256 sps, 0.1 sps battery, 32 sps DRL/REF.

Availability: Muse 2016 ( MU_02 ) only

IXNMusePresetPreset23 

6 CH. Factory usage only. Commercial headbands do not support this preset.

Availability: Muse 2016 ( MU_02 ) only

IXNMusePresetPresetAb 

6 CH, 16 bit @ 500 sps, 50 sps accelerometer, 0.1 sps battery, compression OFF, notch filter OFF, no error data, no DRL/REF data

Research preset:
Only the following data packets are generated with this preset: IXNMuseDataPacketTypeEeg, IXNMuseDataPacketTypeAccelerometer, IXNMuseDataPacketTypeBattery
Artifacts are not generated with this preset.

Availability: Muse 2014 ( MU_01 ) only

IXNMusePresetPresetAd 

4 CH, 16 bit @ 500 sps, 50 sps accelerometer, 0.1 sps battery, compression OFF, notch filter OFF, no error data, no DRL/REF data

Research preset:
Only the following data packets are generated with this preset: IXNMuseDataPacketTypeEeg, IXNMuseDataPacketTypeAccelerometer, IXNMuseDataPacketTypeBattery
Artifacts are not generated with this preset.

Availability: Muse 2014 ( MU_01 ) only
