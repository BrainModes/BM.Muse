Detailed Description

A File interface. Normally you will not need to use this class directly as you can get a IXNMuseFileReader or IXNMuseFileWriter using IXNMuseFileFactory
If you provide your own implementation of this class, you should provide a way to specify the path to the file you want to access in a platform specific way.

Threading IXNMuseFile is not thread safe by default, but IXNMuseFileWriter provides the necessary synchronization for its usage. If you are using IXNMuseFile outside of that context, make sure the calls to the open: (IXNMuseFile-p), write: (IXNMuseFile-p) and close: (IXNMuseFile-p) functions are thread-safe.
