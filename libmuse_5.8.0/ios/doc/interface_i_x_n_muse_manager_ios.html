- (void) showMusePickerWithCompletion: (EABluetoothAccessoryPickerCompletion)  completion

Displays an alert that allows the user to pair the device with a Muse.

This just calls showBluetoothAccessoryPickerWithNameFilter:completion: with a filter that passes devices starting with "Muse".

Deprecated:
This will only display Muse 2014 ( MU_01 ) headsets. Instead set a IXNMuseListener with setMuseListener: (IXNMuseManager-p) to receive callbacks when either model of headband is detected. See the sample application for an example of how to update the UI and connect to the headset.
